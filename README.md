# 🖨 인쇄소 가이드 (Printshop Guide)

인쇄소 정보를 한 번에 모아보는 큐레이션 가이드 사이트입니다.  
**서버 없이 순수 HTML/CSS/JS**로 동작하며, 데이터는 `data/` 폴더의 JS 파일만 수정하면 됩니다.

---

## 📁 파일 구조

```
index.html          ← 메인 HTML (탭 구조)
style.css           ← 전체 스타일 (CSS 변수 기반)
app.js              ← 앱 로직 (필터·검색·큐레이션·렌더링)
data/
  printers.js       ← 인쇄소 데이터 (PRINTERS_DATA 전역 변수)
  designers.js      ← 디자이너 데이터 (DESIGNERS_DATA 전역 변수)
```

---

## ✅ 구현된 기능

| 기능 | 설명 |
|------|------|
| 🌟 큐레이션 탭 | 12가지 카테고리 카드 → 클릭 시 해당 필터로 인쇄소 탭 이동 |
| 🔍 통합 검색 | 히어로 검색 + 인쇄소 목록 검색 (이름·지역·태그 모두 검색) |
| 🏷️ 다중 필터 | 인쇄 방식 / 제품 유형 / 특징 3개 그룹 독립 필터 |
| 🔀 정렬 | 기본순 / 이름순 / 추천(⭐) 먼저 |
| 📋 인쇄소 목록 | 전체 목록 카드 뷰 + 모달 상세 보기 |
| ✏️ 디자이너 탭 | 전문 분야 필터 + 검색 + 포트폴리오/이메일 링크 |
| 📖 인쇄 가이드 | 인쇄 방식·용지·후가공·파일 준비·제본·꿀팁 6개 섹션 |
| 📊 통계 바 | 등록 인쇄소·추천 수·지역 수·디자이너 수 애니메이션 표시 |
| 📱 반응형 | 모바일·태블릿·PC 모두 대응 |
| ♿ 접근성 | ARIA 레이블, 시맨틱 HTML, 키보드 탐색 지원 |

## 🛠 데이터 수정 방법 (서버 불필요)

### 인쇄소 추가/수정
`data/printers.js` 파일의 `PRINTERS_DATA.printers` 배열에 항목 추가:

```js
{
  "name": "인쇄소 이름",          // ⭐️ 포함 시 추천 인쇄소로 표시
  "website": "https://...",
  "sns": "https://instagram.com/...",  // 없으면 ""
  "location": "서울특별시 마포구",
  "tags": ["디지털 인쇄", "책자", "소량 인쇄"]  // 아래 태그 목록 참조
}
```

**사용 가능한 태그 목록:**
- 인쇄 방식: `디지털 인쇄`, `옵셋 인쇄`, `리소 인쇄`
- 제품 유형: `책자`, `리플렛`, `명함`, `포스터`, `스티커`, `굿즈`, `카드 및 엽서`, `라벨`, `달력`
- 특징: `소량 인쇄`, `대량 인쇄`, `당일 인쇄`, `박 후가공`, `양장 제본`, `간편 견적`, `독립출판`, `디자이너 친화`

### 디자이너 추가/수정
`data/designers.js` 파일의 `DESIGNERS_DATA.designers` 배열에 항목 추가:

```js
{
  "name": "디자이너/스튜디오 이름",
  "email": "email@example.com",
  "portfolio": "https://...",
  "tags": ["단행본", "리플렛"],
  "intro": "소개글"
}
```

### 큐레이션 카테고리 추가/수정
`app.js` 파일 상단의 `CURATED_CATEGORIES` 배열에 항목 추가:

```js
{
  id: 'unique-id',
  icon: '🖨️',
  title: '카드 제목',
  desc: '카드 설명',
  filterTags: ['태그명'],  // 위 태그 중 하나
}
```

## 🗂 탭 구조 및 URL 파라미터

현재 SPA(Single Page Application) 구조로 탭 전환은 JS로 동작합니다.

| 탭 ID | 내용 |
|--------|------|
| `curated` | 추천/큐레이션 (기본 탭) |
| `printers` | 인쇄소 전체 목록 + 필터·검색 |
| `designers` | 디자이너 목록 + 필터·검색 |
| `guide` | 인쇄 기초 가이드 |

## 🎨 디자인 커스터마이징

`style.css` 상단 `:root` 변수 수정으로 전체 색상·스타일 변경 가능:

```css
:root {
  --accent: #E63946;     /* 포인트 컬러 */
  --primary: #1a1a1a;    /* 주요 색상 (헤더·버튼 등) */
  --bg: #f7f7f5;         /* 배경색 */
}
```
